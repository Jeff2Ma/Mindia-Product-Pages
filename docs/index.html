Installation

Install using FTP

  • Unzip the basix.zip file locally to your machine
  • Connect to your FTP area using your preferred FTP package
  • Upload the basix folder that you extracted from the zip file to the themes folder of your WordPress installation (wp-content -> themes)
  • Go to Appearance -> Themes and and activate Basix.

Install through WordPress

  • Login to your WordPress admin area
  • Navigate to Appearance -> Themes and click the Add New button at the top
  • Click the Upload Theme button at the top
  • Click the Choose File button and locate the basix.zip file on your machine and click the Install Now button
  • On the basis the theme installs correctly, click the Activate link

Now that you have activated the theme, you can install the included plugins which are as follows:

  • Visual Composer (Required)
  • Icon Manager (Required)
  • Revolution Slider
  • Basix Twitter Feed

You should see a notice once you have installed the theme telling you that the theme requires the plugins mentioned above. Simply click on the 'Begin installing plugins' link, check the box at the top of the table to select all of them, click the bulk actions dropdown and choose 'Install'. Now click Apply. I have also included the Simple Page Sidebars plugin as a recommended plugin. If you don't want to use this plugin then make sure it's not checked.

If you choose not to install the plugins at this stage, you can install them later by going to Appearance -> Install Plugins and install from there.

Demo Data Import

Blog Posts and Content Pages Demo Import

To import the same content as used in the Basix theme demo that you will have already seen, follow these steps:

  • Navigate to Tools -> Import
  • Choose WordPress and install it if it isn't already installed
  • Click the 'Activate Plugin and run Importer' if you had to install it
  • Click the 'Choose File' button
  • Locate the 'basix-demo-content.xml' file from the Demo Content folder and click the 'Upload file and import' button
  • Assign the posts to your existing admin user from the dropdown
  • Select the 'Download and import file attachments' checkbox
  • Click the 'Submit' button

You can now go to settings and select one of the home pages as your static front page and 'Blog' as your posts page. While you're there, if you have activated the Simple Page Sidebars plugin, you can select the 'Blog Sidebar' as your default sidebar. This is required for the plugin to work correctly.

Note: The default post 'Hello Word' should be removed otherwise you'll have a post shown in the latest news carousel without an image and may look at bit odd to start with.

Revolution Slider Import

There's a good chance you won't want all of the included slider demos so just pick the one that suits you best to get started.

  • Navigate to Revolution Slider in the admin menu
  • Click the green Import Slider button then click the Choose File button
  • Locate one of the demo sliders that were included in the main zip file package e.g. rev_slider_home_standard.zip
  • Click Import Slider

Menu Setup

If you've imported the demo data then you'll want to associate the menus from that data import with the correct menu locations. Navigate to Appearance -> Menus and set the Footer Menu as the Footer Navigation and click the Save Menu button. Now select the Main Menu from the dropdown at the top of the page and check the Primary Navigation checkbox at the bottom. Again, click the Save Menu button.

If you are creating your site from scratch then you need to create your menus as you would any other menu in WordPress and associate them with either the 'Primary Navigation' or the 'Footer Navigation'. Any other menus you would like can be inserted with widgets using the WordPress Widgets under Appearance or Visual Composer.

Wide Menu Format

To use the wide menu format simply set the parent menu item class to "wide" along with how many columns you would like e.g. "col-5". So combined, the CSS Classes field for your parent menu item should consist of "wide col-5". All menu items under this parent menu item will be split up into columns and ordered by the menu item order.

Mega Menu Format

To use the mega menu format which allows column organised content you need to use the class "mega" along with how many columns e.g. "col-4". Now you can add parent columns simply by adding menu items (link) using "#!" as the URL. Now you can add children menu items that will be listed in their respective column. See the demo content menus for clarification.

Portfolio

To create a portfolio, simply go to the Portfolio menu and add your items. Custom categories are supported and allow the end user to filter your portfolio with clean animation using jQuery MixItUp. You can either allow the theme to control the portfolio detail page layout or check the "Hide featured image on actual page" checkbox and you can use Visual Composer and change the layout to your heart's content! The featured image will still be used on the main portfolio page which can be created using either the 3 or 4 column portfolio templates.

Importing the demo content will reveal how this all comes together but it's very simple. Create a page using the portfolio template then add your items using the Portfolio Menu. Then either use the default portfolio detail page layout or disable the featured image and create your own layout!

Visual Composer

Visual Composer is a drag and drop frontend and backend page builder plugin that will save you time working on the site content. You will be able to take full control over your WordPress site, build any layout you can imagine â€“ no programming knowledge required. There are two ways in which you can work with Visual Composer, using the Frontend Editor or using the Backend Editor. I am personally a fan of the Backend Editor however the Frontend Editor will allow you to to see your changes live (within reason). Try out both and see which you prefer! For more information on how to use Visual Composer, please refer to the Visual Composer specific documentation included with your download.

Revolution Slider

I have added some custom styles and some clever jQuery to achieve the left aligned slide style that works responsively with multiple break points. As standard, Revolution Slider doesn't allow you to set the slider the full width and auto responsive and keep layers aligned to the left side of your content area. Well, thanks to me, it does now! All layers styled with a class that contains "left_aligned" will automatically align to the left of your site content container across all screen resolutions.

For further information on how to use Revolution Slider, please refer to the specific Revolution Slider documentation included.
